ESCAPE GAME - CARTE INTERACTIVE

Instructions :

1. Pour modifier les images :
   - Remplace les fichiers dans le dossier /images avec tes propres images.
   - Les noms doivent correspondre : paris.jpg, londres.jpg, moscou.jpg, etc.

2. Pour mettre en ligne :
   - Va sur https://app.netlify.com/drop
   - Glisse-dépose le dossier complet (avec index.html, style.css, script.js, images/)
   - Tu obtiendras une URL publique immédiatement.

3. Pour générer un QR code :
   - Va sur https://www.qr-code-generator.com/
   - Colle l’URL Netlify et télécharge ton QR code.

Amuse-toi bien !
